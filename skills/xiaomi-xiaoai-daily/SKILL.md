# 小爱同学用户反馈日报 Skill

> 📌 安装方式：克隆仓库后，将此文件夹复制到 OpenClaw workspace 的 `skills/` 目录下

## 核心功能

1. **多平台搜索**：使用Tavily搜索小爱同学相关用户反馈
2. **正负面分类**：将用户反馈按正面、负面分类梳理
3. **日报生成**：生成结构化的小爱同学用户反馈日报
4. **定时发送**：支持cron定时发送到飞书

## 使用流程

### 手动生成日报

1. 使用 Tavily 搜索工具搜索各平台用户反馈
2. 按正负面分类整理
3. 按标准格式输出日报

### 搜索命令示例

```bash
# 搜索正面评价
tavily-search "小爱同学 好评 体验 超级小爱 好用" --topic general -n 10

# 搜索负面评价
tavily-search "小爱同学 吐槽 差评 翻车 麦克风" --topic general -n 10

# 搜索功能动态
tavily-search "小爱同学 新功能 更新 超级小爱 升级" --topic general -n 10
```

### 定时任务设置（每日早上9点）

```bash
openclaw cron add --name "小爱同学日报" \
  --schedule "0 9 * * *" \
  --payload '{"kind":"agentTurn","message":"生成今日小爱同学用户反馈日报"}' \
  --sessionTarget isolated \
  --delivery '{"mode":"announce","channel":"feishu"}'
```

## 最佳实践

1. **搜索策略**
   - 关键词组合：品牌词+评价词+平台词
   - 优先搜索近期内容（近一周）
   - 多平台交叉验证

2. **内容筛选**
   - 优先选择真实用户反馈
   - 区分个人体验与客观评测
   - 关注重复出现的共性问题

3. **正负面判断**
   - 事实性问题（如麦克风缩水）归为负面
   - 功能赞赏归为正面
   - 功能动态单独分类

4. **格式一致性**
   - 使用标准的日报模板
   - 标注信息来源
   - 保持时间更新准确
